# 人材机判断机制（目标：资料积累后 ≥80% 可自动判断）

> 不能做到 100% 准确；通过**多源融合 + 分项置信度 + 工艺校验**，在样本充足时力争 **≥80%** 的清单项可自动或半自动给出可靠的人材机拆分。

---

## 一、判断什么

对每条清单（名称 + 项目特征 + 单位），输出：

| 输出 | 说明 |
|------|------|
| 主材 / 辅材 / 人工 / 机械 | 元/清单单位 |
| **综合置信度** | 0~1 |
| **分项置信度** | 各列单独可信度 |
| **来源** | line_match / craft_profile / material_plus_template / … |
| **预警** | 如「涂料项人工为 0」 |

---

## 二、四层判断（优先级从高到低）

```text
① 整项历史匹配（名称+特征+做法标签均达标）
        ↓ 未达 80% 置信
② 工艺价型库 craft_cost_profiles（同工艺×同城×同档，样本≥3 更稳）
        ↓
③ 主材价库 + 工艺份额模板（主材准、辅材/人工按历史份额推算）
        ↓
④ 不填或弱参考（<55%），备注需人工组价
```

| 来源 | 典型置信度 | 何时自动填入 |
|------|------------|--------------|
| `line_match` | 0.80~0.98 | 名称+特征+做法均相似 |
| `market_reference` | 0.82 | `config/market_reference_prices.json` 公开市场/信息价 |
| `craft_profile` | ≤0.74 | 同工艺样本≥3且非「装饰综合」、通过合理性校验 |
| `material_plus_template` | 0.68~0.78 | 有规格主材价 + 工艺模板 |
| `line_match_weak` | 0.55~0.79 | 导出时参考填入并标注 |
| `unresolved` | <0.55 | 不自动填 |

阈值：`config/settings.json` → `component_judge.auto_fill_confidence`（默认 **0.8**）。

---

## 三、工艺类型识别

`config/craft_trade_rules.json` 定义装饰/机电工艺（涂料、块料、隔墙、吊顶、电缆、灯具等）。

- `classify_craft(名称, 特征, 单位)` → 工艺 id + 应有主辅材结构  
- 学习资料每增加 → `learn` / `backfill-kb` 自动重建 **工艺价型库**

---

## 四、合理性校验

按工艺检查：

- 涂料项是否有人工、主材  
- 地砖项是否有主材  
- 轻钢隔墙是否以辅材+人工为主  

结合 `material_catalog`（主材辅材判定表）做材料角色提示。

---

## 五、命令

```powershell
# 重建价库（含工艺价型）
python app.py backfill-kb

# 试判一条
python app.py judge "块料楼地面 600*600" --feature "1.水泥砂浆粘贴 2.10厚地砖" --unit ㎡ --city 深圳 --tier mid

# 批量回测准确率（对比历史人材机拆解，默认导出 Excel）
python app.py audit-judge
python app.py audit-judge --limit 500          # 快速抽检
python app.py audit-judge --include-self       # 含同项匹配（会高估，仅对比用）
```

导出：`data/exports/抽检报告_人材机判断.xlsx`（明细 / 汇总 / 按来源 / 按工艺）。

### 回测指标说明

| 指标 | 含义 |
|------|------|
| **auto_fill_rate** | 判断器认为置信度≥80%、可自动填入的比例 |
| **core_main_and_labor** | 主材、人工均在容忍范围内（分项±20% 或绝对±5元，合计±15%） |
| **accuracy_auto_fill_only** | 仅在「可自动填」子集上的准确率（是否真达到 80% 目标） |
| **exclude_self** | 默认排除同标准项，避免整项匹配套自己 |

容忍度可在命令行调整：`--rel-tol 0.20 --total-tol 0.15`。

招标导出已默认走 `judge_line_components`（`inplace_bidder`）。

---

## 六、如何提高准确率

1. **多给带成本拆解的历史 Excel**（`learn`）— 直接抬高 ①② 层命中率  
2. **特征写清做法**（几遍漆、砂浆/瓷砖胶、防水）— 做法标签不冲突  
3. **标注城市/档位** — 价型库按 `city × tier` 分桶  
4. 定期 `backfill-kb` — 聚合 `line_price_facts`、`material_price_facts`、`craft_cost_profiles`  

资料越多，①② 层占比越高，整体越接近 **80%+** 自动判断。
