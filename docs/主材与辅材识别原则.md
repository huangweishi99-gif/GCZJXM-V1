# 主材与辅材识别原则

> 与 `docs/项目特征与施工工艺.md` 配套。

## 国标有没有「主材」「辅材」？

**没有。** GB/T 50500 清单计价标准只要求报**综合单价**及其组成，不强制区分主材、辅材列。  
区分来自：

| 体系 | 常用说法 |
|------|----------|
| **定额** | **计价材料**（定额价内，多当辅材） vs **未计价材料**（另计市场价，安装/装饰里大量主材、设备） |
| **行业通俗** | 价高、量大、重要部位 → 主材；价低、量少、隐蔽 → 辅材（合同约定优先） |
| **投标组价表** | 人工 + 主材 + 辅材 + 机械（企业习惯，非国标强制字段名） |

建标〔2013〕44 号费用组成只列**材料费**，不拆主辅。

## 本项目采用的造价实务口径

## 主材

1. **项目名称里通常已写明** — 清单项名称就是「要做什么」，主材品类往往就在名称中。  
2. **肉眼可见的面层材料** — 工程完工后，业主/审计一眼能认出的那一层。  

| 项目名称示例 | 主材理解 |
|--------------|----------|
| 2.5mm厚铝单板（氟碳喷涂） | 氟碳铝单板面板 |
| 800×800地砖 | 地砖 |
| 内墙乳胶漆 | 乳胶漆（面层涂料） |
| 钢化玻璃隔断 | 钢化玻璃 |

## 辅材

- 服务于**施工步骤**（项目特征）中的安装、粘结、密封、保护。  
- 用量相对小、品种杂，多列在「辅材」或组价表「其他辅材」。  

| 典型辅材 | 说明 |
|----------|------|
| 自攻钉、膨胀螺栓、角码垫块 | 固定用 |
| 泡沫棒、部分密封胶 | 幕墙填缝（个别项目把胶算主材） |
| 电工胶带、接线端子 | 电气安装辅耗 |
| 保护膜、美纹纸 | 成品保护 |

## 与项目特征的分工

```
项目名称  →  做什么 + 主材是什么（面层）
项目特征  →  施工步骤（基层、龙骨、厚度、固定方式…）
主材列    →  面层材料钱
辅材列    →  安装辅耗钱
```

**组价时**：先读名称定主材品类，再逐条读特征定步骤是否一致，最后核对主材/辅材/人工/机械是否匹配历史项目。

## 系统说明

系统**不自动改写** Excel 中的主材/辅材分列，只学习表内已有划分。  

### 招标组价参照顺序（`reference_resolve`）

1. **整项参照**：历史项的**项目名称**与**项目特征**均达到相似度（见 `config/settings.json` → `reference_line`），才套用整条人材机。  
2. **主材参照**：整项不够相似时，从名称/特征识别材料品类与规格（如 `600*600` 地砖），在库内查**相同规格**历史**主材单价**中位数填入（人工/辅材/机械可为 0，备注标注「主材参考」）。  
3. **无参照**：不强行填价，备注说明原因。

规则配置：`config/material_rules.json`（品类关键词、规格正则）。

## 材料判定目录（`主材辅材判定.xlsx`）

导入命令：`python app.py import-catalog`

| 口径 | 说明 |
|------|------|
| **项目实务** | 名称里的面层、完工可见 → **主材**（地砖、涂料、玻璃…） |
| **判定表目录** | 按材料品类建档：龙骨、石膏板、腻子、水泥等归 **辅材**；涂料、瓷砖、不锈钢等归 **主材** |

二者不矛盾：目录是「这种材料在组价表里通常归哪一列」；项目名称仍是判断**本条清单**主材的首选。

**判定表辅材（7 类）**：石膏板、腻子、轻钢龙骨、夹板、水泥、白水泥、防水材料等 — 多对应特征里的基层/骨架工序。

**系统函数**：`classify_material_role(名称, 特征)` 按目录最长匹配，用于备注或后续自动校验主辅材列。

详见 `docs/清单定额与组价原理.md`、`docs/装饰工程施工工艺与定额套用.md`。
