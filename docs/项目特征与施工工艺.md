# 清单组价铁律（项目名称 · 特征 · 主材）

## 国标依据（检索摘要）

| 来源 | 要点 |
|------|------|
| **GB 50500-2013 / GB/T 50500-2024**《建设工程工程量清单计价标准》 | 分部分项清单五要素：**项目编码、项目名称、项目特征、计量单位、工程量** |
| **项目特征**（术语 2.0.7 / 2.0.5） | 构成清单项目**自身价值的本质特征**；确定综合单价不可缺少的依据；特征不符可调价（第 9.4 节） |
| **工作内容**（GB/T 50854-2024 等计量标准） | 完成清单项目的**主要施工工序**；除另有规定外，视为已含完成该项所需的必要辅助工序 |
| **主材 / 辅材** | **国标清单规范不单独定义**；来自**定额计价**习惯（计价材料 / 未计价材料）及行业按价格、用量、是否构成实体划分 |

参考：[GB 50500-2013](http://m.jianbiaoku.com/webarbs/book/104/2116944.shtml)、[GB/T 50500-2024](https://www.gongbiaoku.com/book/o8p264613z7)、[GB/T 50854-2024](https://www.gongbiaoku.com/book/dcm26460lul)

## 三句话定调（造价实务 + 与国标衔接）

| 字段 | 含义 |
|------|------|
| **项目名称** | **要做什么** — 工程内容本身 |
| **项目特征** | **本质特征**（材质、规格、做法、部位等）— 确定「怎么做、做到什么标准」；编制时宜**逐条写清** |
| **工作内容** | （计量标准附录）完成该项的**主要工序**；投标综合单价一般视为已含必要辅工序 |
| **主材** | 一般在**项目名称里就写明了**，且多为完工后**肉眼能看到的面层材料**（行业口径；定额里常对应**未计价材料**） |

名称相同、单位相同，只要**施工步骤（特征）**不同，人工、主材、辅材、机械都可能不同，**绝不能只按名称套价**。

---

## 项目名称 = 做什么

示例：

- `2.5mm厚铝单板（氟碳喷涂）` → 做氟碳铝单板面层  
- `内墙乳胶漆` → 做乳胶漆涂装  
- `落地式配电箱` → 安装该配电箱  

项目名称往往直接带出**主材品类**（面层是什么）。

---

## 项目特征：国标叫「本质特征」，实务常写成施工做法

国标定义是**自身价值的本质特征**（材质、规格、部位、做法等），不是简单重复项目名称。  
国内招标清单里常写成 `1.… 2.… 3.…` 条目，**每一条往往对应一道工序或一道做法要求**——与本系统「特征=步骤」的匹配逻辑一致。

与**工作内容**（计量标准附录列的主要工序）的区别：工作内容偏「清单单价已含哪些活」；特征偏「做成什么样、用什么做」。二者有交叉，编制时避免在特征里重复堆砌已在工作内容中隐含的一般辅工序。

特征示例（不是重复写名称）：

```
1.部位：楼地面面层
2.厚度：20厚以内
3.做法：水泥砂浆随打随抹光，素水泥浆一道
4.综合考虑找坡、配合比…
```

幕墙示例：

```
1.骨架材料种类、规格、中距: 300*200*12mm热浸镀锌钢通
2.面层材料: 2.5mm厚铝单板（氟碳喷涂）
3.面层固定方式: 铝合金垫块、角码、螺栓…
```

**步骤不同 → 工法不同 → 单价不同**，系统按逐条特征比对。

## 主材 vs 辅材（本项目约定）

### 主材

- **与项目名称一致或对应**，是工程**看得见的那一层**  
  例：铝单板、瓷砖、玻璃、乳胶漆、石材、木地板、配电箱（设备类可视为主材/设备）  
- Excel 里通常在 **「主材」「材料费」** 列  
- 组价分析表里材料明细第一行，往往就是面层主材（用量×单价）

### 辅材

- **为主材安装服务**，完工后多**看不见**或不易单独计量  
  例：螺钉、膨胀螺栓、胶垫、密封胶（部分项目把胶算主材）、扎带、保护膜、零星五金  
- Excel 里在 **「辅材」** 列，或分析表里的「其他辅材」

记忆口诀：**名称里写的、完工能看到的面层 → 主材；藏在做法里、用来安装固定的 → 多为辅材。**

## 系统如何实现

| 环节 | 做法 |
|------|------|
| 解析 | `parse_craft_points()` 把特征拆成**施工步骤**列表 |
| 入库 | `method_signature` = 名称 + 单位 + 做法标签 + 步骤指纹 |
| 匹配 | 特征全文 40% + 步骤/标签 45% + 名称 15% |
| 成本列 | 尊重表头「主材/辅材/人工/机械」；有综合单价分析表时优先用拆解价 |
| 冲突 | 步骤对不上或关键做法矛盾 → **不自动套价** |

## 维护
