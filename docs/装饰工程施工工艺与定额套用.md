# 装饰工程施工工艺与定额套用要点

> 用于理解「项目特征」应写哪些工序、以及广联达装饰定额大致对应关系。  
> 具体子目编号以录入 `quota_items` 后的深圳/广联达册为准。

---

## 1. 楼地面工程

| 工序（特征常见写法） | 作用 | 主材/辅材 | 定额方向 |
|----------------------|------|-----------|----------|
| 基层清理、刷界面剂 | 粘结准备 | 辅材 | 找平/铺贴前工序 |
| 水泥砂浆找平（厚度） | 标高、平整 | 辅材（水泥砂） | 楼地面找平 |
| 防水涂膜/卷材 | 湿区防渗 | 辅材/有时单列主材 | 防水 |
| 粘结层（砂浆/瓷砖胶） | 固定面层 | 辅材 | 铺贴 |
| 面层铺贴（砖/石/地板） | 可见面层 | **主材** | 块料/木地板 |
| 勾缝、清洁 | 完工 | 辅材 | 常含在铺贴 |

**单位**：清单多为 **㎡**；定额消耗量表常按 **100㎡**。

---

## 2. 墙柱面工程

| 工序 | 主材/辅材 | 说明 |
|------|-----------|------|
| 基层处理、抹灰（厚度、砂浆标号） | 辅材 | 特征写清厚度 |
| 腻子（遍数、耐水） | 辅材 | 与涂料工序分开 |
| 乳胶漆/无机涂料（颜色、遍数） | **主材** | 名称常带涂料类型 |
| 墙砖/石材湿贴或干挂 | 面层**主材** | 粘结方式必须写清 |

---

## 3. 天棚工程

| 工序 | 主材/辅材 | 说明 |
|------|-----------|------|
| 轻钢龙骨、吊杆 | **辅材**（目录） | 吊顶骨架 |
| 石膏板/硅酸钙板封板 | **辅材**（目录） | 本项目目录归辅材 |
| 腻子、涂料 | 腻子辅材、涂料主材 | 与天棚涂料项对应 |
| **无机涂料** | **与乳胶漆同工序** | 定额册多列「乳胶漆」子目；清单写无机涂料时**套乳胶漆定额/历史乳胶漆价** |
| 铝单板/铝格栅吊顶 | **主材** | 金属面层 |

---

## 4. 幕墙 / 金属 / 玻璃

| 类型 | 特征要点 | 主材 |
|------|----------|------|
| 铝板幕墙 | 板厚、氟碳、龙骨、密封胶 | 铝单板 |
| 玻璃隔断 | 玻璃厚度、型材、五金 | 玻璃 |
| 不锈钢饰面 | 材质、厚度、基层 | 不锈钢 |

---

## 5. 项目特征编写检查单

组价前核对：

- [ ] 是否写清**面层主材**（名称里或特征第 1 条）  
- [ ] 是否写清**粘结/做法**（砂浆 / 瓷砖胶 / 干挂）  
- [ ] 是否写清**厚度**（找平、抹灰、涂层）  
- [ ] 湿区是否写**防水**  
- [ ] 涂料是否写**遍数、底漆**  
- [ ] 单位是否与清单一致（㎡ / m / 樘）

## 7. 后续：定额子目入库后

每条 `quota_items` 含：

- `unit_quota`（如 100㎡）、`unit_scale`（换算系数）  
- `work_content`（工作内容）  
- 人材机消耗量  

匹配时：**清单特征 ↔ 定额工作内容** 相似度达标才推荐套用。
