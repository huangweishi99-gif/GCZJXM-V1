## 一、项目目标（已确认）

### 1.1 核心能力

| 能力 | 说明 |
|------|------|
| **自主学习** | 每次提供新清单 + 内部成本价 → 自动识别表头 → 写入本地知识库 |
| **自主组价** | 甲方招标清单 → 按 **项目名称 + 项目特征 + 单位** 匹配历史 → 生成 **投标方单价分析表** |
| **相同项识别** | **精确匹配** + **模糊匹配** 双模式，可配置阈值与是否自动填价 |
| **智能表头** | 列数可变、备注可选、双行表头、多「合价」列、成本区双主材列 |

### 1.2 业务规则（已确认）

- 管理费 / 利润 / 税金：**暂固定 0% / 0% / 0%**
- 需报价行：**有项目名称 + 有单位 + 有工程量**
- 甲方表列数不固定，以识别结果为准
- 投标方导出：**18 列 (A～R)**，含成本合价列与 Excel 公式链（见 `docs/表头对照与计算公式.md`）

### 1.3 匹配模式定义

| 模式 | 代号 | 规则 | 自动填价 |
|------|------|------|----------|
| **精确匹配** | `exact` | 名称≥98%、特征≥92%、单位一致 | 仅 A 级 |
| **模糊匹配** | `fuzzy` | 名称≥85%、特征≥75%、单位一致 | A、B 级 |
| **自动** | `auto`（默认） | 先按精确找；无 A 则放宽到模糊 | A 自动；B 填价但标记需审核 |

**匹配等级**

| 等级 | 含义 |
|------|------|
| A | 精确相同项 |
| B | 高度相似（模糊） |
| C | 仅部分相似，供参考 |
| D | 未匹配，人工组价 |

## 二、系统架构（当前实现）

```
app.py / ui.py          ← 命令行 / 图形界面
    ├── ingest          ← 智能表头 detector + parser
    ├── knowledge       ← learn 入库、标准项归并
    ├── match           ← 精确/模糊/自动 匹配引擎
    ├── pricing         ← 组价 + 公式链 calc
    └── export          ← 18列投标方 Excel（含公式）

data/cost_pricing.db    ← SQLite 知识库
config/settings.json    ← 阈值、费率、匹配模式
config/column_templates.json
```

## 三、运行环境与依赖（非 Cursor 扩展）

本项目的「安装」指 **Python 依赖**，**不需要**安装特殊 Cursor/VS Code 扩展即可运行。

```bash
pip install -r requirements.txt
```

| pandas, openpyxl, xlrd | Excel 读写 |
| rapidfuzz | 模糊相似度 |

| pywin32 | 可选，Windows 下校验 Excel 公式 |

**资料目录**：`清单数据资料/`、`清单数据资料/AI学习清单/`

**Cursor 项目规则**：`.cursor/rules/工程造价组价.mdc`（AI 协作约定，非运行时依赖）

## 五、使用流程

```
[历史项目 Excel] ──learn──► 知识库
                                │
[甲方招标 Excel] ──tender──► 清单行 ──price──► 组价结果 ──export──► 投标方单价分析表.xlsx
                              （识别表头）      （精确/模糊匹配）
```

**命令示例**

```powershell
python app.py init
python app.py learn "清单数据资料/AI学习清单/某项目.xlsx"
python app.py tender "招标清单.xlsx" --price --export --match-mode auto
python app.py match "轻钢龙骨隔墙" --feature "100mm" --unit "㎡"
streamlit run ui.py
```

## 六、数据模型摘要

- **standard_items**：`(name_norm, feature_norm, unit_norm)` 唯一
- **cost_records**：主材、损耗率、人工、辅材、机械、成本单价、来源项目
- **pricing_lines**：匹配等级 A/B/C/D、置信度、建议单价构成、匹配说明

## 七、风险与对策

| 风险 | 对策 |
|------|------|
| 表头千差万别 | detector 扫描 + 别名表可配置 |
| 误匹配 | 精确/模糊分离；B 级标记需审核 |
| 无成本历史 | D 级导出不强行填数 |
| 特征表述差异大 | rapidfuzz + 可调阈值 |
