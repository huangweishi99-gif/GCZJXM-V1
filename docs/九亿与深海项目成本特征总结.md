### 2.4 特征与做法特点

1. **清单写法**：序号制，**无 12 位国标编码**；特征为多条目编号（1.2.3.）。  
2. **涂料**：大量「墙面乳胶漆 PT-01」「天棚乳胶漆」，特征中**较少写几遍漆**，做法标签常显示「未识别具体做法」——组价时要靠**全文相似 + 您的历史价**，建议在特征中补写「两遍/三遍」。  
3. **贴砖**：如「浅色水磨石砖」→ 识别为 **水泥砂浆粘贴**（特征含 DSM/结合层等）。  
4. **隔墙**：100mm / 200mm / 弧形 分开项，单价 130～221 元/㎡ 不等，**墙厚与弧形必须分项匹配**。  
5. **安装**：强电、电缆、灯具等项多，成本单价高、特征偏设备规格（WDZB1-YJV 等）。

### 2.5 同名不同做法（需分组套价）

表中存在**同名不同单价**项（做法或特征不同），例如多种「墙面乳胶漆」「石材窗台板」在不同房间分部重复出现、单价 0 与有数并存（未填或汇总行）。组价时**不能按名称 alone 套价**。

### 3.4 特征与做法特点（本库最适合「做法匹配」）

1. **国标清单编码** + **物料代号**（CT-01、MT-02、CL-01），特征长、含燃烧等级、做法层、品牌要求。  
2. **涂料（无机涂料/乳胶漆）**：特征中明确 **「三遍涂刷」** 等 → 系统识别为 **乳胶漆三遍**，与两遍可区分。  
3. **贴砖做法区分清晰**：  
   - 地砖 CT-01 → **水泥砂浆粘贴**（DSM15、找平等）  
   - 墙面 CT-04 → **瓷砖胶粘贴**  
   - 有水区 CT-03 → 砂浆粘贴 + **湿区** + 防火 A 级  
4. **不锈钢门套/门**：MT-02 喷砂香槟金，特征含 1.2 厚、五金、门号，单价约 492～981 元/㎡。  
5. **防火等级**：大量 **A 级 / B1 级** 标注，匹配时应作为关键标签。  
6. **安装 Sheet**：灯具、开关、线缆等，成本单价已解析项较少，多数行需对照原表补读。

## 七、组价使用建议

1. **新招标清单若接近深海格式**：优先用深海库匹配，特征含「三遍漆、瓷砖胶、防火 A」时命中率高。  
2. **若接近九亿格式**：重点核对 **隔墙厚度、玻璃规格、防水、强电**；乳胶漆建议在特征中补写遍数。  
3. **两库同时存在时**：按 `match-mode auto`，做法冲突则不自动套价，见 `docs/做法特征匹配说明.md`。
